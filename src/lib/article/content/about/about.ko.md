## Glitch Text Generator 에 오신 것을 환영합니다

Glitch Text Generator 는 **33 가지의 독특한 텍스트 스타일 변환**을 제공하는 강력하고 완전히 무료인 온라인 도구입니다. 유니코드 문자 변환의 마법을 통해 소셜 미디어, 게임, 채팅 애플리케이션에서 메시지를 돋보이게 하는 다양한 멋지고 독특한 텍스트 효과를 쉽게 만들 수 있습니다.

## 우리는 누구인가

우리는 유니코드 문자 예술과 텍스트 변환 기술에 열정을 가진 개발 팀입니다. 우리는 텍스트가 단순히 정보의 매체가 아니라 창의성과 개성을 표현하는 도구라고 믿습니다. 그래서 누구나 일반 텍스트를 놀라운 시각 효과로 쉽게 변환할 수 있도록 이 플랫폼을 만들었습니다.

## 우리의 미션

- **텍스트 변환을 간단하고 쉽게** - 복잡한 작업이 필요 없이 텍스트를 입력하면 즉시 결과를 얻을 수 있습니다
- **다양한 애플리케이션 시나리오 지원** - Discord, Roblox, Minecraft 와 같은 인기 플랫폼과 호환됩니다
- **다국어 지원 제공** - 전 세계 사용자를 위한 10 개 언어 인터페이스 지원
- **완전히 무료이고 개방적** - 모든 기능이 영구적으로 무료이며 등록이나 결제가 필요하지 않습니다

## 핵심 기능

### 🎭 효과 생성기

다양한 특수 효과 생성기를 제공합니다:

- **Zalgo 텍스트** - 유니코드 결합 기호를 사용하여 무섭고 손상되고 괴짜 스타일의 텍스트 효과 생성
- **저주받은 텍스트** - 통화 기호와 라틴 변형을 사용하여 불안한 텍스트 생성
- **뒤집힌 텍스트** - 텍스트를 수평 또는 수직으로 반전
- **해커 텍스트** - 매트릭스 스타일 터미널 텍스트 효과
- **숨겨진 텍스트** - 텍스트를 완전히 보이지 않게 만들지만 공간은 차지

### 🔤 유니코드 글꼴 변형

유니코드 수학 영숫자 기호를 기반으로 23 가지 글꼴 스타일을 제공합니다:

- 굵게, 기울임꼴, 굵은 기울임꼴
- 스크립트, 프락투르
- 산세리프, 모노스페이스
- 이중 선, 원형 문자
- 사각형 문자, 전각 문자
- 그 외 많은 것들...

### 🎮 플랫폼 전용 생성기

특정 플랫폼에 최적화된 텍스트 효과:

- **Discord 글리치 텍스트** - Discord 사용자 이름 및 메시지와 완벽하게 호환
- **Roblox 글리치 텍스트** - 모바일 및 데스크톱 Roblox 에 적응
- **Minecraft 글리치 텍스트** - Java 및 Bedrock 에디션 지원

## 기술적 하이라이트

우리 웹사이트는 현대적인 기술 스택으로 구축되었습니다:

- **순수 유니코드 변환** - 모든 효과는 유니코드 문자를 통해 달성되며 이미지나 특수 CSS 가 필요하지 않습니다
- **정적 사이트 생성** - 초고속 페이지 로딩과 뛰어난 성능
- **반응형 디자인** - 데스크톱, 태블릿 및 모바일 장치에 완벽하게 적응

## 우리를 선택하는 이유

✅ **즉시 변환** - 실시간 미리보기, 대기 없음  
✅ **개인정보 보호** - 모든 변환은 브라우저에서 로컬로 수행되며 텍스트를 업로드하지 않습니다  
✅ **크로스 플랫폼 호환성** - 생성된 텍스트는 유니코드를 지원하는 모든 플랫폼에서 작동합니다  
✅ **지속적인 업데이트** - 새로운 텍스트 스타일과 기능을 지속적으로 추가합니다  
✅ **오픈소스 친화적** - 표준 유니코드 사양을 기반으로 투명하고 신뢰할 수 있습니다
