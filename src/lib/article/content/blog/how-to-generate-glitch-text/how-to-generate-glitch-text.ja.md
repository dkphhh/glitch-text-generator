## 方法 1：Zalgo テキストジェネレーター（最も古典的）

### Zalgo テキストとは？

**Zalgo テキスト**は最も古典的なグリッチテキストエフェクトで、文字の**上下と中央**に多数の Unicode 結合記号を重ねることで、不気味で壊れた視覚効果を生み出します。

**視覚的特徴**：

- 文字の上下に多くの「触手のような」記号がある
- 「侵食された」または「壊れた」ように見える
- 高度に調整可能（1〜10 の強度レベル）
- 非常にインパクトがあり、怖い

### 技術原理

Zalgo テキストは**Unicode 結合発音記号**（U+0300〜U+036F）を使用し、これらは基本文字の上に重ねることができる記号です：

```
基本文字：H
上部記号を追加：H̴̡̢̛̛͖͉̰
下部記号を追加：H̴̡̢̛̛͖͉̰̩̹̫̠̻̪̫̲
中央記号を追加：H̴̡̢̛̛͖͉̰̩̹̫̠̻̪̫̲̺̜̺̼̳
```

生成アルゴリズム：

1. 各文字にランダムに 0〜N 個の上部記号を追加
2. 各文字にランダムに 0〜N 個の下部記号を追加
3. 各文字にランダムに 0〜N 個の中央記号を追加
4. N は強度レベル（1〜10）によって決定

### 使用例

✅ **適している場合**：

## 方法 2：グリッチフォントジェネレーター（最も実用的）

### グリッチフォントとは？

**グリッチフォント**は、通常の文字を**視覚的に類似した**Unicode 文字に置き換え、「壊れた」または「歪んだ」エフェクトを作成しながら**基本的な読みやすさを維持**します。

**視覚的特徴**：

- 文字が「置き違えられた」または「置き換えられた」ように見える
- 元のテキストの輪郭を維持
- Zalgo より読みやすい
- 長いテキストに適している

### 技術原理

グリッチフォントは**Unicode 文字マッピングテーブル**を使用し、各ラテン文字を視覚的に類似した Unicode 文字にマッピングします：

```typescript
マッピング例：
A → Ѧ、₳
B → ฿、Ƀ、ᗾ
E → Ξ、Σ、€、Є
H → Ħ
O → Ø、Ө、Ѳ、Θ
```

これらの置換文字は以下から来ています：

- **キリル文字**：Ѧ、Б、Д
- **ギリシャ文字**：Ξ、Σ、Θ
- **通貨記号**：฿、₳、€
- **ラテン拡張**：Ħ、Ø、Ƶ

### 使用例

✅ **適している場合**：

- ニックネームとユーザー名
- ブランドアイデンティティ（非公式）
- ソーシャルメディアの自己紹介
- ゲームギルド名
- タイトルとスローガン

✅ **利点**：

- 良好な互換性（ほぼすべてのプラットフォームでサポート）
- 読みやすさを維持
- 検索で使用可能（一部のプラットフォーム）
- フィルタリングシステムに誤判定されない

## 技術的深さ：なぜコピー＆ペーストできるのか？

### Unicode 統一標準

**Unicode とは？**

- 149,000 以上の文字を含む
- 159 の文字体系をカバー
- すべての現代的なシステムでサポート

**結合記号の魔法**：
Unicode は「結合文字」メカニズムを設計し、1 つの文字が複数の修飾子と重ねることができます：

```
基本：e
+ 結合記号1：é（U+0301 鋭アクセント）
+ 結合記号2：ė（U+0307 上点）
+ 結合記号3：ę（U+0328 オゴネク）
結果：ę́̇（3つの記号が重なる）
```

Zalgo テキストはまさにこのメカニズムを使用し、数十の結合記号を重ねます。

## よくある質問

### Q: グリッチテキストエフェクトは安全ですか？

**A**：完全に安全です。標準の Unicode 文字変換のみを使用し、データを収集または保存せず、すべての処理はブラウザでローカルに行われます。

### Q: なぜ正常に表示されないことがあるのですか？

**A**：考えられる理由：

1. フォントがその Unicode 文字をサポートしていない
2. デバイスまたはアプリのバージョンが古すぎる
3. プラットフォームが特殊文字を制限またはフィルタリング
