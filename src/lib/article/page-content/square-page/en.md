The squared text generator creates a modern minimalist decorative effect where each character is wrapped in a square frame. This effect is achieved through enclosed alphanumeric symbols in Unicode.

## Square Enclosed Characters in Unicode

Unicode's enclosed alphanumeric symbols block includes not only circular characters but also square (rectangular) enclosed characters. These characters were originally designed for numbering, notation, and special typographic needs.

Square enclosed characters have the following features:

- Each character is surrounded by a complete square frame
- Frames are typically right-angled borders
- Characters are centered within the square
- Present a modern, stable visual style

## Design Language of Squares

In design psychology, squares and circles convey different visual information:

- **Circles**: Soft, friendly, flowing
- **Squares**: Stable, modern, professional

Squared characters give text a more modern, rational, and professional visual temperament through square frames.

Squared and [circled](/generator/circle-text-generator) characters each have their characteristics:
