## 核心功能

### 🎭 特效生成器

我们提供多种特殊效果生成器，包括：

- **Zalgo 文本** - 使用 Unicode 组合符号创建恐怖、腐蚀和极客风格的文本效果
- **诅咒文本** - 使用货币符号和拉丁变体创造令人不安的文字
- **翻转文本** - 水平或垂直镜像你的文字
- **黑客文本** - Matrix 风格的终端文本效果
- **隐形文本** - 让文字完全不可见但占据空间

### 🔤 Unicode 字体变体

基于 Unicode 数学字母数字符号，我们提供 23 种字体样式：

- 粗体、斜体、粗斜体
- 手写体、哥特体
- 无衬线、等宽字体
- 空心字体、圆圈字母
- 方框字母、全角字符
- 以及更多...

### 🎮 平台专属生成器

针对特定平台优化的文本效果：

- **Discord 故障文本** - 完美兼容 Discord 用户名和消息
- **Roblox 故障文本** - 适配移动端和桌面端 Roblox
- **Minecraft 故障文本** - 支持 Java 版和基岩版
