글리치 텍스트 효과는 시각적으로 "손상되고", "왜곡되어" 또는 "으스스한" 것처럼 보이는 텍스트 스타일로, 일반적으로 다음에서 볼 수 있습니다:

- **공포 게임** 인터페이스 텍스트
- **사이버펑크 스타일** 디자인
- **소셜 미디어** 개인화된 닉네임
- **Discord/게임** 캐릭터 이름
- **예술 창작** 및 포스터 디자인

## 방법 1: Zalgo 텍스트 생성기 (가장 클래식)

### Zalgo 텍스트란?

**Zalgo 텍스트**는 가장 클래식한 글리치 텍스트 효과로, 문자 **위, 아래, 중간에** 수많은 Unicode 결합 마크를 쌓아 으스스하고 손상된 시각 효과를 만듭니다.

**시각적 특징**:

- 문자 위아래에 많은 "촉수 같은" 기호가 있음
- "침식되거나" "손상된" 것처럼 보임
- 높은 조정 가능성 (1-10 강도 레벨)
- 극도로 임팩트 있고 무서움

### 기술 원리

Zalgo 텍스트는 **Unicode 결합 발음 부호** (U+0300–U+036F) 를 사용하며, 이는 기본 문자에 쌓이도록 설계된 기호입니다:

```
기본 문자: H
위 마크 추가: H̴̡̢̧̛̛͖͉̰
아래 마크 추가: H̴̡̢̧̛̛͖͉̰̩̹̫̠̻̪̫̲
중간 마크 추가: H̴̡̢̧̛̛͖͉̰̩̹̫̠̻̪̫̲̺̜̺̼̳
```

생성 알고리즘:

1. 각 문자에 0-N 개의 위 마크를 무작위로 추가
2. 각 문자에 0-N 개의 아래 마크를 무작위로 추가
3. 각 문자에 0-N 개의 중간 마크를 무작위로 추가
4. N 은 강도 레벨 (1-10) 에 의해 결정됨

### 사용 사례

✅ **적합한 경우**:

- 공포/스릴러 테마 콘텐츠
- Discord 서버 이름
- 게임 캐릭터 이름 (일부 게임 지원)
- 예술 창작
- 밈 제작

❌ **부적합한 경우**:

- 공식 비즈니스 사용
- 명확한 읽기가 필요한 시나리오
- 일부 게임이 필터링하거나 금지할 수 있음

**단계**:

1. 우리의 [Zalgo 텍스트 생성기](/generator/zalgo-text-generator) 방문
2. 텍스트 입력
3. 강도 슬라이더 조정 (1-10 레벨)
4. "복사" 버튼 클릭
5. 어디든지 붙여넣기

**강도 권장 사항**:

- **레벨 1-3**: 약간의 효과, 가독성 유지
- **레벨 4-6**: 중간 효과, 시각과 인식의 균형
- **레벨 7-10**: 극단적 효과, 완전한 손상 느낌

## 방법 2: 글리치 폰트 생성기 (가장 실용적)

### 글리치 폰트란?

**글리치 폰트**는 일반 문자를 **시각적으로 유사한** Unicode 문자로 대체하여 "손상되거나" "왜곡된" 효과를 만들면서 **기본 가독성을 유지**합니다.

**시각적 특징**:

- 문자가 "잘못 배치되거나" "대체된" 것처럼 보임
- 원래 텍스트 윤곽 유지
- Zalgo 보다 더 읽기 쉬움
- 긴 텍스트에 적합

### 기술 원리

글리치 폰트는 **Unicode 문자 매핑 테이블**을 사용하여 각 라틴 문자를 시각적으로 유사한 Unicode 문자로 매핑합니다:

```typescript
매핑 예시:
A → Ѧ, ₳
B → ฿, Ƀ, ᗾ
E → Ξ, Σ, €, Є
H → Ħ
O → Ø, Ө, Ѳ, Θ
```

이러한 대체 문자는 다음에서 나옵니다:

- **키릴 문자**: Ѧ, Б, Д
- **그리스 문자**: Ξ, Σ, Θ
- **통화 기호**: ฿, ₳, €
- **라틴 확장**: Ħ, Ø, Ƶ

### 사용 사례

✅ **적합한 경우**:

- 닉네임 및 사용자 이름
- 브랜드 아이덴티티 (비공식)
- 소셜 미디어 바이오
- 게임 길드 이름
- 제목 및 슬로건

✅ **장점**:

- 좋은 호환성 (거의 모든 플랫폼 지원)
- 가독성 유지
- 검색에 사용 가능 (일부 플랫폼)
- 필터링 시스템에 의해 잘못 판단되지 않음

## 기술 깊이: 복사 및 붙여넣기가 가능한 이유는?

### Unicode 통합 표준

**Unicode 란?**

- 149,000 개 이상의 문자 포함
- 159 개의 문자 체계 포함
- 모든 현대 시스템이 지원

**결합 마크의 마법**:
Unicode 는 "결합 문자" 메커니즘을 설계하여 하나의 문자가 여러 수정자와 쌓일 수 있도록 합니다:

```
기본: e
+ 결합 마크 1: é (U+0301 예각 악센트)
+ 결합 마크 2: ė (U+0307 위 점)
+ 결합 마크 3: ę (U+0328 오고넥)
결과: ę́̇ (세 개의 마크가 쌓임)
```

Zalgo 텍스트는 정확히 이 메커니즘을 사용하여 수십 개의 결합 마크를 쌓습니다.
