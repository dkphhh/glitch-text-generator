### 什么是 Unicode？

Unicode 是一个国际标准字符集，包含超过 14 万个字符，涵盖世界上几乎所有的文字系统、符号和表情符号。关键是：

✅ **所有现代平台都支持**：Windows、Mac、iOS、Android、Web
✅ **纯文本格式**：可以直接复制粘贴

✅ **跨应用兼容**：Discord、Roblox、Minecraft、社交媒体全支持

### Unicode 如何创造文字乱码效果？

Unicode 中有几类特殊字符可以用来创建乱码效果：

1. **组合字符（Combining Characters）**
   - 这些字符可以"附加"在基础字母上下左右
   - 例如：̈ ̃ ̀ ́ ̂ ̄ ̆ ̇ ̊ ̋ ̌ ̍ ̎ ̏
   - 通过叠加多个组合字符，创造 Zalgo 混乱效果

2. **字符变体（Character Variants）**
   - Unicode 包含多种字母变体
   - 例如：𝕳𝖊𝖑𝖑𝖔（哥特体）、𝓗𝓮𝓵𝓵𝓸（草书体）
   - 直接替换普通字母即可

3. **旋转字符（Rotated Characters）**
   - 某些语言的字符看起来像英文字母的旋转版
   - 例如：ɐ ǝ ɟ ɥ ı ɾ ʇ ʌ ʍ ʎ
   - 用于创建翻转文字效果

4. **装饰符号（Decorative Symbols）**
   - 框线、圆圈等装饰符号
   - 例如：🄰🄱🄲（方框字母）、①②③（圆圈数字）
