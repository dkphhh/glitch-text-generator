## 方法一：Zalgo 文字生成器（最经典）

### 什么是 Zalgo 文字？

**Zalgo 文字**是最经典的文字错乱效果，通过在字符**上方、下方和中间**叠加大量 Unicode 组合符号（combining diacritical marks），创造出诡异、损坏的视觉效果。

**视觉特点**：

- 字符上下有大量"触手"般的符号
- 看起来像被"侵蚀"或"损坏"
- 高度可调节（1-10 级强度）
- 极具冲击力和恐怖感

### 技术原理

Zalgo 文字使用 **Unicode 组合变音符号**（Combining Diacritical Marks，U+0300–U+036F），这些符号设计用于叠加在基础字符上：

```
基础字符：H
添加上方符号：H̴̡̢̛̛͖͉̰
添加下方符号：H̴̡̢̛̛͖͉̰̩̹̫̠̻̪̫̲
添加中间符号：H̴̡̢̛̛͖͉̰̩̹̫̠̻̪̫̲̺̜̺̼̳
```

生成算法：

1. 为每个字符随机添加 0-N 个上方符号
2. 为每个字符随机添加 0-N 个下方符号
3. 为每个字符随机添加 0-N 个中间符号
4. N 由强度等级（1-10）决定

## 方法二：故障字体生成器（最实用）

### 什么是故障字体？

**故障字体**（Glitch Font）通过将普通字符替换为**视觉上相似**的 Unicode 字符，创造出"损坏"或"扭曲"的效果，同时**保持基本可读性**。

**视觉特点**：

- 字符看起来"错位"或"替换"
- 保持原始文字的大致轮廓
- 比 Zalgo 更易阅读
- 适合长文本

### 技术原理

故障字体使用 **Unicode 字符映射表**，将每个拉丁字母映射到外观相似的 Unicode 字符：

```typescript
映射示例：
A → Ѧ, ₳
B → ฿, Ƀ, ᗾ
E → Ξ, Σ, €, Є
H → Ħ
O → Ø, Ө, Ѳ, Θ
```

这些替换字符来自：

- **西里尔字母**（Cyrillic）：Ѧ, Б, Д
- **希腊字母**（Greek）：Ξ, Σ, Θ
- **货币符号**（Currency）：฿, ₳, €
- **拉丁扩展**（Latin Extended）：Ħ, Ø, Ƶ

✅ **适合**：

- 品牌标识（非正式）

✅ **优势**：

- 保持可读性

## 技术深度：为什么可以复制粘贴？

### Unicode 统一标准

**Unicode 是什么？**

- 包含超过 149,000 个字符
- 覆盖 159 种文字系统
- 被所有现代系统支持

**组合符号的魔力**：
Unicode 设计了"组合字符"机制，允许一个字符与多个修饰符叠加：

```
基础：e
+ 组合符号 1：é（U+0301 锐音符）
+ 组合符号 2：ė（U+0307 点）
+ 组合符号 3：ę（U+0328 反尾）
结果：ę́̇（三个符号叠加）
```

Zalgo 文字正是利用这个机制，叠加多达几十个组合符号。
