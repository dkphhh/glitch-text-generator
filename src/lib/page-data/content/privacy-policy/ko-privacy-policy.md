**발효일: 2025 년 11 월 16 일**

Glitch Text Generator(이하 "웹사이트" 또는 "당사") 에 오신 것을 환영합니다. 당사는 귀하의 개인정보 보호를 매우 중요하게 생각합니다. 본 개인정보 보호정책은 당사가 귀하의 정보를 수집, 사용, 저장 및 보호하는 방법을 설명합니다.

## 1. 정보 수집 개요

### 1.1 당사가 수집하지 않는 정보

**웹사이트의 약속:**

- ❌ **입력한 텍스트 콘텐츠를 저장하지 않음**: 텍스트 생성기에 입력하는 모든 콘텐츠는 브라우저에서만 로컬로 처리되며 당사 서버에 업로드되지 않습니다
- ❌ **등록 또는 로그인 불필요**: 계정을 만들지 않고도 모든 기능을 사용할 수 있습니다
- ❌ **개인 식별 정보를 수집하지 않음**: 이름, 이메일, 전화번호 또는 기타 개인 정보를 수집하지 않습니다
- ❌ **개인 데이터 저장에 쿠키를 사용하지 않음**: 쿠키를 통해 개인 식별 정보를 저장하지 않습니다

### 1.2 당사가 수집하는 익명 데이터

서비스 품질과 사용자 경험을 개선하기 위해 제 3 자 분석 도구를 통해 다음과 같은 익명 통계 정보를 수집합니다:

**Google Analytics 를 통해 수집:**

- 페이지 조회수 및 탐색 경로
- 사용 장치 유형 (데스크톱, 모바일, 태블릿 등)
- 브라우저 유형 및 버전
- 운영 체제 정보
- 지리적 위치 (국가/지역 및 도시 수준)
- 트래픽 소스 (검색 엔진, 직접 액세스, 외부 링크 등)
- 페이지 체류 시간
- 사용된 생성기 유형 (Zalgo, Cursed, Flip 등)

**Microsoft Clarity 를 통해 수집:**

- 익명 세션 재생 (사용자 상호작용 패턴 분석용)
- 히트맵 데이터 (클릭, 스크롤 동작)
- 페이지 성능 지표
- JavaScript 오류 로그

**중요 사항:** 이러한 데이터는 모두 익명이며 귀하의 개인 신원과 연결되지 않습니다. 이 데이터를 통해 특정 개별 사용자를 식별할 수 없습니다.

## 2. 데이터 처리 방법

### 2.1 로컬 처리

모든 텍스트 변환 기능은 브라우저에서 로컬로 완료됩니다:

- **유니코드 문자 변환**: 순수 프론트엔드 JavaScript 를 사용하여 구현되며 서버 개입이 필요하지 않습니다
- **실시간 미리보기**: 모든 미리보기 효과는 클라이언트 측에서 렌더링됩니다
- **텍스트 복사**: 복사 기능은 브라우저의 기본 클립보드 API 를 사용합니다

**기술 구현:**

- 수학적 영숫자 기호 변환을 위해 `unicode-text-decorator` 라이브러리 사용
- 문자 난독화를 위해 `confusables` 라이브러리 사용
- 특수 효과를 위한 사용자 정의 문자 매핑 사전 (Zalgo, Cursed, Glitch 등)

즉, 텍스트 콘텐츠가 장치를 벗어나지 않으며 당사는 입력한 내용을 보거나 저장할 수 없습니다.

### 2.2 서버 저장소 없음

웹사이트는 정적 사이트 생성 (SSG) 을 사용하여 Cloudflare 에 배포됩니다:

- 웹사이트는 사전 렌더링된 정적 HTML 페이지로 구성됩니다
- 백엔드 서버 또는 데이터베이스 없음
- 사용자 데이터를 저장할 가능성 없음

## 3. 제 3 자 서비스

### 3.1 Google Analytics

**목적:** 웹사이트 트래픽 분석 및 사용자 행동 통계

**수집되는 데이터 유형:**

- 익명 방문 통계
- 페이지 성능 지표
- 사용자 장치 및 브라우저 정보
- 지리적 위치 (국가/도시 수준)

**데이터 보관 기간:** Google Analytics 기본 설정에 따라 데이터는 26 개월 동안 보관됩니다

**개인정보 제어:** 다음 방법으로 Google Analytics 추적을 거부할 수 있습니다:

- [Google Analytics 옵트아웃 브라우저 추가 기능](https://tools.google.com/dlpage/gaoptout) 설치
- 브라우저에서 "추적 안 함" 활성화
- 브라우저의 개인정보 보호/시크릿 모드 사용

**자세한 정보:** [Google Analytics 개인정보 보호정책](https://policies.google.com/privacy)

### 3.2 Microsoft Clarity

**목적:** 사용자 경험 분석 및 웹사이트 최적화

**수집되는 데이터 유형:**

- 익명 세션 재생
- 히트맵 및 스크롤 맵
- 클릭 패턴 분석
- 페이지 성능 데이터

**데이터 처리:** Microsoft Clarity 는 개인 식별 정보 (PII) 를 수집하지 않으며 모든 데이터는 익명입니다

**개인정보 보호:** Clarity 는 민감한 정보 (예: 비밀번호 입력 필드 내용) 를 자동으로 마스킹합니다

**자세한 정보:** [Microsoft Clarity 개인정보 보호정책](https://privacy.microsoft.com/privacystatement)

### 3.3 Cloudflare

**목적:** 웹사이트 호스팅 및 콘텐츠 전송 네트워크 (CDN)

**Cloudflare 가 수집할 수 있는 데이터:**

- IP 주소 (보안 보호 및 트래픽 분석용)
- 브라우저 유형 및 버전
- 액세스 타임스탬프
- 요청된 페이지 URL

**보안 기능:**

- DDoS 보호
- SSL/TLS 암호화
- 웹 애플리케이션 방화벽 (WAF)

**자세한 정보:** [Cloudflare 개인정보 보호정책](https://www.cloudflare.com/privacypolicy/)

## 4. 쿠키 사용

### 4.1 당사가 사용하는 쿠키

웹사이트는 기본 기능을 제공하기 위해 최소한의 쿠키를 사용합니다:

**필수 쿠키 (비활성화 불가):**

- 테마 기본 설정 (라이트/다크 모드)
- 언어 선택 기본 설정 (10 개 언어 옵션: en, zh, es, fr, de, ja, ko, ru, pt, id)
- 세션 상태 유지

**분석 쿠키 (선택 사항):**

- Google Analytics 쿠키 (\_ga, \_gid, \_gat)
- Microsoft Clarity 쿠키 (\_clck, \_clsk)

### 4.2 쿠키 관리 방법

브라우저 설정을 통해 쿠키를 관리하거나 삭제할 수 있습니다:

- **Chrome:** 설정 > 개인정보 및 보안 > 쿠키 및 기타 사이트 데이터
- **Firefox:** 옵션 > 개인정보 및 보안 > 쿠키 및 사이트 데이터
- **Safari:** 환경설정 > 개인정보 보호 > 웹사이트 데이터 관리
- **Edge:** 설정 > 개인정보, 검색 및 서비스 > 쿠키 및 사이트 권한

**참고:** 쿠키를 비활성화하면 테마 설정 및 언어 기본 설정과 같은 특정 웹사이트 기능에 영향을 줄 수 있습니다.
