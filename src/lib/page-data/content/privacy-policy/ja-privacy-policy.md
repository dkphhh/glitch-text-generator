**発効日:2025 年 11 月 16 日**

Glitch Text Generator(以下「本ウェブサイト」または「当社」) へようこそ。当社はお客様のプライバシー保護を非常に重視しています。本プライバシーポリシーは、当社がお客様の情報をどのように収集、使用、保存、保護するかを説明します。

## 1. 情報収集の概要

### 1.1 当社が収集しない情報

**本ウェブサイトの約束：**

- ❌ **入力されたテキストコンテンツを保存しない**:テキストジェネレーターに入力されたすべてのコンテンツは、ブラウザ内でローカルに処理されるのみで、当社のサーバーにアップロードされることはありません
- ❌ **登録またはログインを要求しない**:アカウントを作成せずにすべての機能を使用できます
- ❌ **個人識別情報を収集しない**:お名前、メールアドレス、電話番号などの個人情報は収集しません
- ❌ **個人データの保存に Cookie を使用しない**:Cookie を通じて個人を特定できる情報を保存しません

### 1.2 当社が収集する匿名データ

サービス品質とユーザー体験を向上させるため、サードパーティの分析ツールを通じて以下の匿名統計情報を収集します：

**Google Analytics 経由で収集：**

- ページビューと閲覧パス
- 使用デバイスタイプ (デスクトップ、モバイル、タブレットなど)
- ブラウザタイプとバージョン
- オペレーティングシステム情報
- 地理的位置 (国/地域および都市レベル)
- トラフィックソース (検索エンジン、直接アクセス、外部リンクなど)
- ページ滞在時間
- 使用されたジェネレータータイプ (Zalgo、Cursed、Flip など)

**Microsoft Clarity 経由で収集：**

- 匿名セッションリプレイ (ユーザーインタラクションパターンの分析用)
- ヒートマップデータ (クリック、スクロール動作)
- ページパフォーマンス指標
- JavaScript エラーログ

**重要な注意事項：**これらのデータはすべて匿名であり、お客様の個人的アイデンティティとリンクされていません。これらのデータを通じて特定の個人ユーザーを識別することはできません。

## 2. データ処理方法

### 2.1 ローカル処理

すべてのテキスト変換機能はブラウザ内でローカルに完了します：

- **Unicode 文字変換**:純粋なフロントエンド JavaScript を使用して実装され、サーバーの関与は不要
- **リアルタイムプレビュー**:すべてのプレビュー効果はクライアント側でレンダリング
- **テキストコピー**:コピー機能はブラウザのネイティブクリップボード API を使用

**技術実装：**

- 数学的英数字記号変換に`unicode-text-decorator`ライブラリを使用
- 文字の難読化に`confusables`ライブラリを使用
- 特殊効果用のカスタム文字マッピング辞書 (Zalgo、Cursed、Glitch など)

つまり、テキストコンテンツがデバイスから外に出ることはなく、当社は入力内容を見たり保存したりすることはできません。

### 2.2 サーバーストレージなし

本ウェブサイトは Static Site Generation(SSG) を使用して Cloudflare にデプロイされています：

- ウェブサイトは事前レンダリングされた静的 HTML ページで構成
- バックエンドサーバーやデータベースなし
- ユーザーデータを保存する可能性なし

## 3. サードパーティサービス

### 3.1 Google Analytics

**目的：**ウェブサイトトラフィック分析とユーザー行動統計

**収集されるデータタイプ：**

- 匿名訪問統計
- ページパフォーマンス指標
- ユーザーデバイスとブラウザ情報
- 地理的位置 (国/都市レベル)

**データ保持期間：**Google Analytics のデフォルト設定により、データは 26 か月間保持されます

**プライバシーコントロール：**以下の方法で Google Analytics のトラッキングをオプトアウトできます：

- [Google Analytics オプトアウト ブラウザアドオン](https://tools.google.com/dlpage/gaoptout)のインストール
- ブラウザで「トラッキング拒否」を有効化
- ブラウザのプライバシー/シークレットモードの使用

**詳細情報：**[Google Analytics プライバシーポリシー](https://policies.google.com/privacy)

### 3.2 Microsoft Clarity

**目的：**ユーザー体験分析とウェブサイト最適化

**収集されるデータタイプ：**

- 匿名セッションリプレイ
- ヒートマップとスクロールマップ
- クリックパターン分析
- ページパフォーマンスデータ

**データ処理：**Microsoft Clarity は個人識別情報 (PII) を収集せず、すべてのデータは匿名です

**プライバシー保護：**Clarity は機密情報 (パスワード入力フィールドの内容など) を自動的にマスクします

**詳細情報：**[Microsoft Clarity プライバシーポリシー](https://privacy.microsoft.com/privacystatement)

### 3.3 Cloudflare

**目的：**ウェブサイトホスティングとコンテンツ配信ネットワーク (CDN)

**Cloudflare が収集する可能性のあるデータ：**

- IP アドレス (セキュリティ保護とトラフィック分析用)
- ブラウザタイプとバージョン
- アクセスタイムスタンプ
- リクエストされたページ URL

**セキュリティ機能：**

- DDoS 保護
- SSL/TLS暗号化
- Web アプリケーションファイアウォール (WAF)

**詳細情報：**[Cloudflare プライバシーポリシー](https://www.cloudflare.com/privacypolicy/)

## 4. Cookie の使用

### 4.1 当社が使用する Cookie

本ウェブサイトは基本機能を提供するために最小限の Cookie を使用します：

**必須 Cookie(無効化不可):**

- テーマ設定 (ライト/ダークモード)
- 言語選択設定 (10 言語オプション:en、zh、es、fr、de、ja、ko、ru、pt、id)
- セッション状態の維持

**分析 Cookie(オプション):**

- Google Analytics Cookie(\_ga、\_gid、\_gat)
- Microsoft Clarity Cookie(\_clck、\_clsk)

### 4.2 Cookie の管理方法

ブラウザ設定から Cookie を管理または削除できます：

- **Chrome:**設定 > プライバシーとセキュリティ > Cookie と他のサイトデータ
- **Firefox:**オプション > プライバシーとセキュリティ > Cookie とサイトデータ
- **Safari:**環境設定 > プライバシー > Web サイトデータを管理
- **Edge:**設定 > プライバシー、検索、サービス > Cookie とサイトのアクセス許可

**注意：**Cookie を無効にすると、テーマ設定や言語設定などの特定のウェブサイト機能に影響する場合があります。

## 5. データセキュリティ

### 5.1 技術的保護措置

お客様の情報を保護するために以下のセキュリティ対策を実施しています：

- **HTTPS 暗号化：**すべてのデータ送信は SSL/TLS で暗号化
- **Cloudflare CDN:**DDoS 保護と Web アプリケーションファイアウォールを提供
- **静的サイトアーキテクチャ：**データベースやバックエンドがなく、データ侵害リスクを軽減
- **ローカル処理：**テキスト変換はブラウザ内で完全に実行され、データはデバイスから外に出ません

### 5.2 サードパーティサービスのセキュリティ

当社が使用するサードパーティサービス (Google Analytics、Microsoft Clarity、Cloudflare) はすべて業界をリードするプロバイダーであり、厳格なセキュリティ基準とプライバシー保護対策を備えています。

## 6. 児童のプライバシー保護

本ウェブサイトは 13 歳未満の児童から故意に個人情報を収集しません。保護者または後見人の方で、お子様が当社に個人情報を提供したことが判明した場合は、当社までご連絡ください。直ちに関連情報を削除いたします。

## 7. 国際的なデータ転送

本ウェブサイトは Cloudflare のグローバル CDN ネットワークを通じて提供され、お客様のデータは異なる国/地域のサーバー間で転送される可能性があります。すべてのデータ転送が適用されるデータ保護法に準拠していることを保証します。

**適用地域：**

- EU ユーザー:一般データ保護規則 (GDPR) に準拠
- 米国ユーザー:カリフォルニア州消費者プライバシー法 (CCPA) に準拠
- その他の地域：適用される地域のデータ保護法に準拠

## 8. お客様の権利

### 8.1 データアクセスとコントロール

当社はお客様の個人データを収集または保存しないため：

- ✅ **アクセス権：**アクセスできるお客様の個人データはありません
- ✅ **削除権：**入力されたテキストは保存されず、削除の必要はありません
- ✅ **データポータビリティ権：**生成されたすべてのテキストは自由にコピーして使用できます
- ✅ **異議申し立て権：**ブラウザ設定から分析 Cookie を無効化できます

### 8.2 分析データのコントロール

以下の方法で分析データ収集をコントロールできます：

1. **Google Analytics を無効化：**
   - Google Analytics オプトアウトブラウザ拡張機能をインストール
   - ブラウザの「トラッキング拒否」機能を有効化

2. **Microsoft Clarity を無効化：**
   - ブラウザで clarity.ms ドメインをブロック
   - プライバシー/シークレットモードでブラウジング

3. **Cookie をクリア：**
   - 定期的にブラウザ Cookie をクリア
   - ブラウザのプライバシー強化機能を使用
